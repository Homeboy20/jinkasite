## How to Use

### Step 1: Access Theme Settings
1. Go to `/admin/login` and log in
2. Click **Settings** in the sidebar
3. Click the **üé® Theme Customization** tab

### Step 2: Customize Your Theme
Choose from these options:

#### Colors
- **Brand Colors**: Primary, Secondary, Accent
- **Status Colors**: Success, Warning, Error
- **Text Colors**: Primary text, Secondary text, Links
- **Backgrounds**: Page background, Card backgrounds
- **Borders**: Border color

#### Styling
- **Button Roundness**: Make buttons square (0) or pill-shaped (50)
- **Card Roundness**: Adjust card corner roundness

#### Fonts
- Choose from 12+ professional fonts
- Separate fonts for headings and body text

### Step 3: Preview Changes
- Watch the **Live Preview** section update as you make changes
- Preview shows exactly how your changes will look

### Step 4: Save
- Click **üíæ Save Theme Settings**
- Changes apply immediately to your entire site
- Cache is automatically cleared

## Quick Tips

### For a Modern Look
- Primary: Bright blue (#3b82f6)
- Border Radius: 12px buttons, 16px cards
- Font: Inter or Poppins

### For a Professional Look
- Primary: Dark blue (#2563eb)
- Border Radius: 6px buttons, 8px cards
- Font: Roboto or Open Sans

### For an Elegant Look
- Primary: Purple (#7c3aed)
- Heading Font: Playfair Display (serif)
- Body Font: Merriweather (serif)

## Examples

### Change Primary Button Color
1. Find "Primary Color" under Brand Colors
2. Click the blue square
3. Pick your new color
4. See buttons update in preview
5. Click Save

### Change Fonts
1. Scroll to "Typography" section
2. Select "Body Font Family" dropdown
3. Choose a font (e.g., "Poppins")
4. Select "Heading Font Family"
5. Choose a heading font
6. See preview update
7. Click Save

### Make Buttons More Rounded
1. Find "Button Border Radius" under Styling Options
2. Increase the number (try 16 for very round)
3. Watch preview buttons change
4. Click Save

## Troubleshooting

**Q: Changes not showing on website?**
- Hard refresh: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
- Try incognito/private window
- Wait 5 seconds and refresh again

**Q: Want to go back to original?**
- Click "üîÑ Reset to Defaults" button

**Q: Color picker not working?**
- Type hex code directly in text field
- Use format: #RRGGBB (e.g., #3b82f6)

**Q: Preview not updating?**
- Save and refresh the settings page
- Check if JavaScript is enabled
